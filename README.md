Опис

У даному проєкті реалізовано гру на пам’ять (Memory Card Game). Гравець відкриває картки та намагається знайти однакові пари. Якщо відкриті картки співпадають, вони залишаються відкритими. Якщо картки різні, вони перевертаються назад.

Функціонал

ігрове поле з картками

відкриття двох карток

перевірка співпадіння

автоматичне перевертання карток назад

перемішування карток при старті

підрахунок ходів

таймер гри

різні рівні складності
